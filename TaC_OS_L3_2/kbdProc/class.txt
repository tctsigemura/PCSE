scConv.cmm:スキャンコードから文字コードに変換する機能を提供するサブ・パッケージ
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
スキャンコード文字コード変換クラス

名称：ScConv
目的：シフトキー状態を合成したスキャンコードを入力し、
      対応する文字コードに変換する。

属性：-scTbl               char[]       スキャンコードと文字コードの対応表
操作：+scConv(char c)      char         スキャンコードから文字コードに変換する。
		  	   		入力(c)はスキャンコードの値にシフトキー
					が押されている場合は0x0100を加えた値。
					適切な文字が無い場合は0x0000を返す。

 kbd.cmm : キーボードドライバ・サブ・パッケージ
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
スキャンコード取得クラス

名称：Kbd
目的：キーボードのスキャンコード(以下ではコード)を取得する。
      但し、(1) 離されたときのコードは返さない。
      	    (2) 複数バイトで表現されるコードは返さない。
            (3) シフトキーのコードは返さないで内部状態を変化させる。
	    (4) 返すコードにシフトキーの状態を合成する(0x0100)。
属性：-rShift              boolean      右シフトキーが押されている
      -lShift              boolean      左シフトキーが押されている
      -stat                int		状態マシンの状態
      -code                char         入力されたコードのバッファ
      			   		(割り込みハンドラが書き込む)
      -sem                 int          割り込みルーチンと同期するセマフォ
操作：-kbdFSM(char c)      char         ハードから取得したコードを入力し、
					  (1) 離されたときのコード除去
      		  	   		  (2) シフトキー操作の認識
					  (3) シフトキー状態とコードを合成出力
					を行う状態マシン。出力無しでは0を返す。
      -kbdInt()            interrupt    割込みハンドラ。kbdFSM を使用し変換後
      			   		スキャンコードをバッファへ書き込む。
      +kbdInit()           void         初期化(ハード、割込みベクタ初期化)
      +getKeyCode()        char		バッファからコードを取り出し返す
					返すコードが無い場合は0x0000を返す。

kbdMain.cmm : CLI 版 MP3 プレーヤ・アプリ・プロセスのサンプル・サブ・パッケージ
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
MP3 プレーヤ・アプリ・プロセス・クラス

名称：KbdMain
目的：MP3 プレーヤアプリのサンプルを提供する．
      このサンプルは音楽再生・停止の CLI 操作ができる他に，HDD内のファイル
      一覧の表示や内容の16進ダンプもできる．
属性：-win                 WIN          CLI が使用する画面ウインド
      -lbuf                char[]       キーボードから１行入力するバッファ
      -kbdCnt              int          lbuf 中の文字数
      ... 以下省略 ....
操作：-getc()              char         キーボードから１文字入力する
      -getLBuf             void         lbuf にキーボードから１行入力する
      +kbdProcInit         void         プロセスの初期化ルーチン
      +kbdMain             void         プロセスのメインルーチン
      ... 以下省略 ....
